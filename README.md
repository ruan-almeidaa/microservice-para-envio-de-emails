#
O e-mail recebe um **JSON** com:

 - "ownerRef": nome de quem está enviando;
 - "emailFrom": e-mail de quem está enviando;
 - "emailTo":  e-mail que vai receber;
 -  "subject":  o assunto do e-mail;
 -  "text": o texto/corpo do e-mail.
